## Wat biedt Epistola jou?

**Duurzaam verdienmodel.** Jij levert diensten (SLA/hosting/support); wij beheren IP + governance. Meerdere leveranciers = marktwerking.

---

## Verdienmodel (realistisch)

| Inkomstenbron | Prijs | Marge |
|---|---|---|
| **Licentie-doorbelasting** | €3–30k/gemeente | Max. 15% (klein) – 10% (groot) |
| **SLA / LCM** | €20–40k/gemeente | Vrije markt |
| **Hosting** | €10–30k/gemeente | Vrije markt |
| **Support / Training** | Variabel | Vrije markt |
| **Custom development** | Variabel | Vrije markt |
| **Features (publiek)** | Betaald door stichting | - |

**Praktijkvoorbeeld gemeente 50k inwoners:**
```
Licentie:           €8.000
  └─ Jij verdient marges: max. 12% = €960 (rest gaat naar stichting)
SLA/hosting:        €35.000 (jij bepaalt)
Support/training:   €5.000 (jij bepaalt)
─────────────────────────
Totaal:             €48.000 (jij factureert aan gemeente)
Jouw inkomsten:     ~€40k (SLA + hosting + support + licentie-marge)
```

## Spelregels (korte versie)

✅ **Mag je doen:**
- Diensten aanbieden (SLA, hosting, support, training, advies)
- Marktprijzen voor services bepalen
- Investeren in doorontwikkeling (features, modules)
- In meerdere aanbestedingen inschrijven

❌ **Mag je niet doen:**
- Lock-in creëren (data portability verplicht)
- Propriëtaire forks (alles upstream)
- Licentie-eisen omzeilen (BSL naleving)
- Exclusieve features (mag niet)

---

## Preferred Supplier status

**Voordelen:**
- Marketing-rechten ("Preferred Supplier by Epistola")
- Prominentie op website/aanbestedingen
- Prioriteit in gezamenlijke communicatie

**Vereisten (ISO/IEC 27001):**
- Certificering + jaarlijks audit
- Upstream bijdragen (20% dev-uren of €10k/jaar)
- SLA-naleving (99%+ uptime)
- Security patches < 48 uur

**Kosten:** €2–5k certificering, €1k/jaar hercertificering

---

## Opstartfase (Oprichters-voordeel)

Huidge leverancier (Zoef.app) heeft tijdelijk voordeel:
- Hoogte-afstemming op roadmap
- Gezamenlijke PR/marketing

**Maar:** Dit is **niet permanent** en geen exclusiviteit. Na jaar 2–3 kan ander leverancier intrede doen.

---

## Roadmap & Investeringen

Stichting investeert in:
- Security, architecture, onderhoud, compliance
- Publieke features (geen vendor-lock)

**Jij kunt investeren in:**
- Klantspecifieke features
- Innovatie
- Integraties

**Regel:** Alles gaat upstream. Geen exclusieve forks.

**Meer informatie:** [Development & Roadmap](/meedoen/leveranciers/roadmap-en-doorontwikkeling)

## Waarborgen voor jou

| Risico | Garantie |
|--------|----------|
| Lock-in door stichting | Nee; stichting levert geen diensten |
| Willekeurige marge-verlaging | Nee; jaarlijks herijking, minimale 30 dagen voormeld |
| IP-geschillen | Nee; all work-for-hire, upstream is contractueel |
| Plotseling ontslagen | Nee; 30 dagen voormeld + transition period |
| Geen inzicht in routekaart | Nee; jaarlijks strategie-sessies |
